Introduction

Design Lite (tiaweb) chip component is a small, interactive element. Chips are commonly used for contacts, text, rules, icons, and photos.

TO INCLUDE AN tiaweb CHIP COMPONENT:

 1. Create a container element for the chip; typically <span> and <div> are used, but any container element should work equally well. If you need interactivity, use a <button>, or add the tabindex attribute to your container.

<span>
</span>

 2. Add in the text wrapper and the tiaweb classes.

<span class="tiaweb-chip">
    <span class="tiaweb-chip__text">Chip Text</span>
</span>

 3. For deletable chips, add in the delete icon. This can be an <a>, <button> or non-interactive tags like <span>.

<span class="tiaweb-chip">
    <span class="tiaweb-chip__text">Chip Text</span>
    <a href="#" class="tiaweb-chip__action"><i class="material-icons">cancel</i></a>
</span>

 4. Contact chips need to have the tiaweb-chip--contact class added to the container, along with another container for the contact icon. The icon container for photos is typically an <img> tag, but other types of content can be used with a little extra supporting css.

<span class="tiaweb-chip">
    <span class="tiaweb-chip__contact tiaweb-color--teal tiaweb-color-text--white">A</span>
    <span class="tiaweb-chip__text">Chip Text</span>
    <a href="#" class="tiaweb-chip__action"><i class="material-icons">cancel</i></a>
</span>

CSS Classes

tiaweb Class Effect Remarks
tiaweb-chip Defines element as an tiaweb chip container Required on "outer" container
tiaweb-chip--contact Defines an tiaweb chip as a contact style chip Optional, goes on "outer" container
tiaweb-chip__text Defines element as the chip's text Required on "inner" text container
tiaweb-chip__action Defines element as the chip's action Required on "inner" action container, if present
tiaweb-chip__contact Defines element as the chip's contact container Required on "inner" contact container, if the tiaweb-chip--contact class is present on "outer" container

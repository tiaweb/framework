Button

TIAWEB menyediakan berbagai class CSS untuk menerapkan berbagai peningkatan visual dan perilaku yang telah ditentukan ke tombol. Tabel berikut mencantumkan class yang tersedia dan efeknya. 

<!-- Raised button -->
<button class="tiaweb-button tiaweb-js-button tiaweb-button--raised">
  Button
</button>
<!-- Raised button with ripple -->
<button class="tiaweb-button tiaweb-js-button tiaweb-button--raised tiaweb-js-ripple-effect">
  Button
</button>
<!-- Raised disabled button -->
<button class="tiaweb-button tiaweb-js-button tiaweb-button--raised" disabled>
  Button
</button>

<!-- Colored raised button -->
<button class="tiaweb-button tiaweb-js-button tiaweb-button--raised tiaweb-button--colored">
  Button
</button>
<!-- Accent-colored raised button -->
<button class="tiaweb-button tiaweb-js-button tiaweb-button--raised tiaweb-button--accent">
  Button
</button>
<!-- Accent-colored raised button with ripple -->
<button class="tiaweb-button tiaweb-js-button tiaweb-button--raised tiaweb-js-ripple-effect tiaweb-button--accent">
  Button
</button>

<!-- Flat button -->
<button class="tiaweb-button tiaweb-js-button">
  Button
</button>
<!-- Flat button with ripple -->
<button class="tiaweb-button tiaweb-js-button tiaweb-js-ripple-effect">
  Button
</button>
<!-- Disabled flat button -->
<button class="tiaweb-button tiaweb-js-button" disabled>
  Button
</button>

Contoh

Berikut akan membantu Anda memahami penggunaan class button untuk menampilkan berbagai jenis button.

1. Button dengan efek "raised" 
<button class="tiaweb-button tiaweb-js-button tiaweb-button--raised">Save</button>
2. Button dengan efek "fab" 

<button class="tiaweb-button tiaweb-js-button tiaweb-button--fab">OK</button>
2. Button dengan efek "colored"

<button class="tiaweb-button tiaweb-js-button tiaweb-button--icon tiaweb-button--colored">?</button>

PILIHAN KONFIGURASI


No. Nama Kelas Deskripsi
1

tiaweb-button

Menetapkan tombol sebagai komponen tiaweb, diperlukan.

2

tiaweb-js-button

Menambahkan perilaku dasar tiaweb ke tombol, diperlukan.

3

(tidak ada )

Mengatur efek tampilan datar ke tombol, default.

4

tiaweb-button--raised

Menyetel efek tampilan terangkat; ini saling eksklusif dengan fab, mini-fab, dan ikon.

5

tiaweb-button--fab

Mengatur efek tampilan hebat (melingkar); ini eksklusif satu sama lain dengan mengangkat, mini-fab, dan ikon.

6

tiaweb-button--mini-fab

Menyetel efek tampilan mini-fab (lingkaran kecil fab); ini saling eksklusif dengan tanda terangkat, luar biasa, dan ikon.

7

tiaweb-button--icon

Mengatur efek tampilan ikon (lingkaran kecil); ini saling eksklusif dengan mengangkat, hebat, dan mini-hebat.

8

tiaweb-button--colored

Menetapkan efek tampilan berwarna di mana warna didefinisikan dalam material.min.css.

9

tiaweb-button--primary

Menetapkan efek tampilan warna primer di mana warna didefinisikan dalam material.min.css.

10

tiaweb-button--accent

Menetapkan efek tampilan warna aksen di mana warna didefinisikan dalam material.min.css.

11

tiaweb-js-ripple-effect

Menetapkan efek klik riak, dapat digunakan bersama dengan kelas lain mana pun.

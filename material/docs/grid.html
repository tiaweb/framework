Grid adalah komponen untuk meletakkan konten untuk berbagai ukuran layar. 

Grid didefinisikan dan ditutup oleh elemen wadah / div. Kotak memiliki 12 kolom di layar ukuran desktop, 8 di layar ukuran tablet, dan 4 di layar ukuran ponsel, di mana setiap ukuran memiliki margin yang telah ditentukan. Grid diletakkan secara berurutan dalam satu baris, sesuai urutannya, dengan pengecualian sebagai berikut: 

 Jika Grid tidak cocok untuk baris dalam salah satu ukuran layar, itu mengulir ke baris berikutnya. 

 Jika Grid memiliki ukuran kolom yang ditentukan sama dengan atau lebih besar dari jumlah kolom untuk ukuran layar, itu mengambil seluruh baris. 

 Tiaweb menyediakan berbagai class CSS untuk menerapkan berbagai peningkatan visual dan perilaku yang telah ditentukan

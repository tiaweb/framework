Grid
Grid adalah komponen untuk meletakkan konten untuk berbagai ukuran layar. 

Grid didefinisikan dan ditutup oleh elemen wadah / div. Kotak memiliki 12 kolom di layar ukuran desktop, 8 di layar ukuran tablet, dan 4 di layar ukuran ponsel, di mana setiap ukuran memiliki margin yang telah ditentukan. Grid diletakkan secara berurutan dalam satu baris, sesuai urutannya, dengan pengecualian sebagai berikut: 

 Jika Grid tidak cocok untuk baris dalam salah satu ukuran layar, itu mengulir ke baris berikutnya. 

 Jika Grid memiliki ukuran kolom yang ditentukan sama dengan atau lebih besar dari jumlah kolom untuk ukuran layar, itu mengambil seluruh baris. 

 Tiaweb menyediakan berbagai class CSS untuk menerapkan berbagai peningkatan visual dan perilaku yang telah ditentukan

Contoh berikut akan membantu Anda memahami penggunaan kelas tiaweb-grid untuk tata letak konten di berbagai layar.

  • tiaweb-layout__header-row - Mengidentifikasi wadah sebagai baris tiaweb header.

  • tiaweb-layout__drawer - Mengidentifikasi div sebagai laci tata letak tiaweb.

  • tiaweb-layout-title - Mengidentifikasi teks judul tata letak.

  • tiaweb-navigation - Mengidentifikasi div sebagai grup navigasi tiaweb.

  • tiaweb-navigation__link - Mengidentifikasi jangkar sebagai tautan navigasi tiaweb.

  • tiaweb-layout__content - Mengidentifikasi div sebagai konten tata letak tiaweb.

  • tiaweb-grid - Mengidentifikasi div sebagai komponen grid tiaweb.

  • tiaweb-cell - Mengidentifikasi div sebagai sel tiaweb.

  • tiaweb-cell - 1-col - Menetapkan ukuran kolom untuk sel menjadi 1 sel dari 12 sel dalam ukuran layar desktop.

  • tiaweb-cell - 4-col - Menetapkan ukuran kolom untuk sel menjadi 4 sel dari 12 sel dalam ukuran layar desktop.

  • tiaweb-cell - 6-col - Menetapkan ukuran kolom untuk sel menjadi 6 sel dari 12 sel dalam ukuran layar desktop.

  • tiaweb-cell - 4-col-phone - Menetapkan ukuran kolom untuk sel menjadi 4 sel dari 4 sel dalam ukuran layar ponsel.

No. Nama class & Deskripsi
1

tiaweb-layout

Mengidentifikasi wadah sebagai komponen tiaweb. Diperlukan pada elemen wadah luar.
2

tiaweb-grid

Mengidentifikasi wadah sebagai komponen kisi tiaweb. Diperlukan pada elemen div "luar".
3

tiaweb-cell

Mengidentifikasi wadah sebagai cell tiaweb. Diperlukan pada elemen div "dalam".
4

tiaweb-grid--no-spacing

Memperbarui cell-cell jaringan untuk tidak memiliki margin di antara mereka. Opsional untuk wadah kotak.
5

tiaweb-cell--N-col

Ini membantu menempatkan ukuran kolom untuk cell ke N, N termasuk 1-12, default ke 4; opsional untuk elemen div "dalam".
6

tiaweb-cell--N-col-desktop

Ini membantu menempatkan ukuran kolom untuk cell ke N dalam mode desktop saja, N adalah 1-12 inklusif; opsional untuk elemen div "dalam".
7

tiaweb-cell--N-col-tablet

Ini membantu menempatkan ukuran kolom untuk cell ke N hanya dalam mode tablet, N adalah 1-8 inklusif; opsional untuk elemen div "dalam".
8

tiaweb-cell--N-col-phone

Ini membantu menempatkan ukuran kolom untuk cell ke N hanya dalam mode telepon, N adalah 1-4 inklusif; opsional untuk elemen div "dalam".
9

tiaweb-cell--hide-desktop

Menyembunyikan cell saat dalam mode desktop. Opsional untuk elemen div "dalam".
10

Tiaweb-cell--hide-tablet

Menyembunyikan cell saat dalam mode tablet. Opsional untuk elemen div "dalam".
11

tiaweb-cell--hide-phone

Menyembunyikan cell saat dalam mode poncell. Opsional untuk elemen div "dalam".
12

tiawweb-cell--stretch

Regangkan cell secara vertikal untuk mengisi induk, default; opsional untuk elemen div "dalam".
13

tiaweb-cell--top

Sejajarkan cell ke atas induk. Opsional untuk elemen div "dalam".
14

tiaweb-cell--middle

Sejajarkan cell ke tengah induk. Opsional untuk elemen div "dalam".
15

tiaweb-cell--bottom

Sejajarkan cell ke bagian bawah induk. Opsional untuk elemen div "dalam".

Tab
Komponen Tab adalah komponen antarmuka pengguna yang membantu menampilkan beberapa layar dalam satu ruang secara eksklusif. 
<div class = "tiaweb-tabs tiaweb-js-tabs">
               <div class = "tiaweb-tabs__tab-bar">
                  <a href = "#tab1-panel" class = "tiaweb-tabs__tab is-active">Tab 1</a>
                  <a href = "#tab2-panel" class = "tiaweb-tabs__tab">Tab 2</a>
                  <a href = "#tab3-panel" class = "tiaweb-tabs__tab">Tab 3</a>
               </div>
            
               <div class = "tiaweb-tabs__panel is-active" id = "tab1-panel">
                  <p>Tab 1 Content</p>
               </div>
            
               <div class = "tiaweb-tabs__panel" id = "tab2-panel">
                  <p>Tab 2 Content</p>
               </div>
            
               <div class = "tiaweb-tabs__panel" id = "tab3-panel">
                  <p>Tab 3 Content</p>
               </div>
            </div>

3. Di dalam div "dalam" pertama (tab), beri kode satu tag <a> (tautan) jangkar untuk setiap tab. Sertakan atribut href dengan nilai untuk mencocokkan nilai atribut id tab, dan beberapa teks tautan singkat. Pada div "batin" yang tersisa (konten), kode id atribut yang nilainya cocok dengan href tautan 
<div>
  <div>
    <a href="#tab1">Tab One</a>
    <a href="#tab2">Tab Two</a>
    <a href="#tab3">Tab Three</a>
  </div>
  <div id="tab1">
  ...
  </div>
  <div id="tab2">

  </div>

  </div>

4. Di dalam div "inner" yang tersisa, kode konten yang ingin Anda tampilkan di setiap panel; gunakan tag HTML standar untuk menyusun konten sesuai keinginan. 
<div>
  <div>
    <a href="#tab1">Tab One</a>
    <a href="#tab2">Tab Two</a>
    <a href="#tab3">Tab Three</a>
  </div>

    <p>Second tab's content.</p>
  </div>
  <div id="tab3">

  </div>

5. Tambahkan satu atau lebih kelas tiaweb, dipisahkan oleh spasi, ke div "luar" dan "dalam" menggunakan atribut kelas; pastikan untuk menyertakan kelas is-active pada tab yang ingin Anda tampilkan secara default. 
<div class="tiaweb-tabs tiaweb-js-tabs">
  <div class="tiaweb-tabs__tab-bar">
    <a href="#tab1" class="tiaweb-tabs__tab">Tab One</a>

Contoh

Berikut akan membantu Anda memahami penggunaan Class tiaweb-tab untuk menata konten pada berbagai tab.

Class tiaweb yang diberikan di bawah ini akan digunakan dalam contoh ini 

  tiaweb-layout - Mengidentifikasi div sebagai komponen tiaweb.
    
  tiaweb-js-layout - Menambahkan perilaku dasar tiaweb ke div luar. 
 
  tiaweb-layout - fixed-header - Membuat header selalu terlihat, bahkan di layar kecil.

  tiaweb-layout__header-row - Mengidentifikasi wadah sebagai baris tajuk tiaweb. 

  tiaweb-layout-title - Mengidentifikasi teks judul tata letak. 

  tiaweb-layout__content - Mengidentifikasi div sebagai konten tata letak tiaweb
 
  tiaweb-tabs - Mengidentifikasi wadah tab sebagai komponen tiaweb. 
 
  tiaweb-js-tabs - Menetapkan perilaku dasar tiaweb ke wadah tab. 

  tiaweb-tabs__tab-bar - Mengidentifikasi wadah sebagai bar tautan tab tiaweb.

  tiaweb-tabs__tab - Mengidentifikasi jangkar (tautan) sebagai aktivator tab tiaweb.

  is-active - Mengidentifikasi tab sebagai tab tampilan default.

  tiaweb-tabs__panel - Mengidentifikasi wadah sebagai konten tab.

PILIHAN KONFIGURASI

No Class Keterangan
1 Tiaweb-layout Mengidentifikasi wadah sebagai komponen tiaweb. Diperlukan pada elemen
2 tiaweb-tab Mengidentifikasi wadah tab sebagai komponen tiaweb. Diperlukan pada elemen div "luar".wadah luar.
3 tiaweb-js-tab tiaweb-js-tab Setel perilaku dasar tiaweb ke wadah tab. Diperlukan pada elemen div "luar".
4 tiaweb-js-ripple-effect Menambahkan efek klik riple ke tautan tab. Pilihan; melanjutkan elemen div "luar".
5 tiaweb-tabs__tab-bar Mengidentifikasi wadah sebagai bilah tautan tab tiaweb. Diperlukan pada elemen div "dalam" pertama.
6 tiaweb-tabs__tab Mengidentifikasi jangkar (tautan) sebagai aktivator tab tiaweb. Diperlukan pada semua tautan di elemen div "dalam" pertama.
7 active Mengidentifikasi tab sebagai tab tampilan default. Diperlukan pada satu (dan hanya satu) elemen "inner" div (tab).
7 tiaweb-tabs__panel Mengidentifikasi wadah sebagai konten tab. Diperlukan pada setiap elemen "tab" dalam (tab).

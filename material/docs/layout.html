Layout
Dalam bab ini, kita akan membahas tata letak yang berbeda di Tiaweb. HTML5 memiliki elemen wadah berikut

  • <div> - Menyediakan wadah umum untuk konten HTML.

  • <header > - Merupakan bagian tajuk.

  • < footer > - Merupakan bagian dari footer.

  • < article > - Merupakan artikel.

  • < section > - Menyediakan wadah umum untuk berbagai jenis bagian.

Fixed Drawer
Untuk membuat templat dengan Fixed/tetap tetapi tanpa header, class Tiaweb berikut digunakan.

  • tiaweb-layout - Mengidentifikasi div sebagai komponen tiaweb.

  • tiaweb-js-layout - Menambahkan perilaku dasar tiaweb ke div luar.

  • tiaweb-layout - fix-drawer - Membuat laci selalu terlihat dan terbuka di layar yang lebih besar.

  • tiaweb-layout__drawer - Mengidentifikasi div sebagai laci tata letak tiaweb.

  • tiaweb-layout-title - Mengidentifikasi teks judul tata letak.

  • tiaweb-navigation - Mengidentifikasi div sebagai grup navigasi tiaweb.

  • tiaweb-navigation__link - Mengidentifikasi jangkar sebagai tautan navigasi tiaweb.

  • tiaweb-layout__content - Mengidentifikasi div sebagai konten tata letak tiaweb.

Sr.No. Class Name & Description
1

tiaweb-layout

Identifies a container as an tiaweb component. Required on outer container element.
2

tiaweb-js-layout

Adds basic tiaweb behavior to layout. Required on outer container element.
3

tiaweb-layout__header

Identifies container as an tiaweb component. Required on header element.
4

tiaweb-layout-icon

Used to add an application icon. Gets overridden by menu icon if both are visible. Optional icon element.
5

tiaweb-layout__header-row

Identifies container as tiaweb header row. Required on header content container.
6

tiaweb-layout__title

Identifies layout title text. Required on layout title container.
7

tiaweb-layout-spacer

Used to align elements inside a header or drawer. It grows to fill remaining space. Commonly used for aligning elements to the right. Optional on div following layout title.
8

tiaweb-navigation

Identifies container as tiaweb navigation group. Required on nav element.
9

tiaweb-navigation__link

Identifies anchor as tiaweb navigation link. Required on header and/or drawer anchor elements.
10

tiaweb-layout__drawer

Identifies container as tiaweb layout drawer. Required on drawer container element.
11

tiaweb-layout__content

Identifies container as tiaweb layout content. Required on main element.
12

tiaweb-layout__header--scroll

Makes the header scroll with the content. Optional on header element.
13

tiaweb-layout--fixed-drawer

Makes the drawer always visible and open in larger screens. Optional on outer container element and not on drawer container element.
14

tiaweb-layout--fixed-header

Makes the header always visible, even in small screens. Optional on outer container element.
15

tiaweb-layout--large-screen-only

Hides an element on smaller screens. Optional on any descendant of tiaweb-layout.
16

tiaweb-layout--small-screen-only

Hides an element on larger screens. Optional on any descendant of tiaweb-layout.
17

tiaweb-layout__header--waterfall

Allows a "waterfall" effect with multiple header lines. Optional on header element.
18

tiaweb-layout__header--transparent

Makes header transparent and draws on top of layout background. Optional on header element.
19

tiaweb-layout__header--seamed

Uses a header without a shadow. Optional on header element.
20

tiaweb-layout__tab-bar

Identifies container as an tiaweb tab bar. Required on container element inside header (tabbed layout).
21

tiaweb-layout__tab

Identifies anchor as tiaweb tab link. Required on tab bar anchor elements.
22

is-active

Identifies tab as default active tab. Optional on tab bar anchor element and associated tab section element.
23

tiaweb-layout__tab-panel

Identifies container as tab content panel. Required on tab section elements.
24

tiaweb-layout--fixed-tabs

Uses fixed tabs instead of the default scrollable tabs. Optional on outer container element , not container inside header.
